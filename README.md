## 📊 Executive Summary

본 문서는 프로관전러 P.S의 데이터 분석 방법론을 기반으로, League of Legends 경기 데이터에서 
라인별로 가장 중요한 10개의 Feature(지표)를 추출한 결과입니다.

- 분석 대상: match_0.json (1경기 데이터)
- 경기 시간: 약 25분 (1504초)
- 분석 방식: PS 스타일 (라인별 가중치 기반)
- 추출된 Feature: 라인별 10개 × 5 라인 = 총 50개
